Calculatrice¶

Documentation de la classe calculator

calculator.add(x, y)¶

Cette fonction additionne deux nombres et retourne la somme de ces derniers.

Paramètres

  • x (int) – Premier nombre

  • y (int) – Second nombre

Renvoie

Somme des deux nommbres

Type renvoyé

int

calculator.subtract(x, y)¶

Cette fonction soustrait deux nombres et retourne la différence de ces derniers.

Paramètres

  • x (int) – Premier nombre

  • y (int) – Second nombre

Renvoie

Différence entre les deux nommbres

Type renvoyé

int

calculator.divide(x, y)¶

Cette fonction divise deux nombres et retourne la quotient de ces derniers.

Paramètres

  • x (int) – Premier nombre

  • y (int) – Second nombre

Renvoie

Quotient des deux nommbres

Type renvoyé

float

calculator.multiply(x, y)¶

Cette fonction multiplie deux nombres et retourne la multiplication de ces derniers.

Paramètres

  • x (int) – Premier nombre

  • y (int) – Second nombre

Renvoie

Multiplication des deux nommbres

Type renvoyé

int
